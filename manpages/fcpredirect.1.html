NAME

fcpredirect - insert a freenet URI, as a redirect to another URI

 

SYNTAX

fcpredirect [options] src_uri target_uri

 

DESCRIPTION

fcpredirect is a simple command-line FCP client program which inserts a URI into freenet, as a redirect to another URI.

Note the order of arguments - you specify the 'source' first, and the 'target' second. This is the opposite of commands such as 'ln'.

 

OPTIONS

-h, --help
Print help information and exit
-V, --version
Print version number and exit
-v, --verbose
Enable very verbose output, including FCP transcripts, to the stderr stream.
-H, --fcpHost=<hostname>
Use FCP interface at host <hostname>, defaults to 127.0.0.1
-P, --fcpPort=<port>
Use FCP interface at port <port>

 

FILES

none
 

ENVIRONMENT VARIABLES

FCP_HOST
Specifies the hostname of the FCP interface. This value will be used unless overridden with '-H'.
FCP_PORT
Specifies the port number of the FCP interface. This value will be used unless overridden with '-P'.

 

EXAMPLES

fcpredirect KSK@glitters KSK@gold

Inserts a new key KSK@glitters, which when retrieved redirects to the key KSK@gold
