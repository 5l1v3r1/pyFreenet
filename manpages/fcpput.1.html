NAME

fcpput - Insert single key, or directory, into Freenet  

SYNTAX

fcpput [options] key_uri [filename]  

DESCRIPTION

fcpput is a simple command-line FCP client program for inserting keys or directories into Freenet.

If the program completes successfully, it will print to standard output a freenet URI under which the key can be later retrieved.

 

OPTIONS

-h, --help
Print help information and exit
-V, --version Print version number and exit
-v, --verbose Enable very verbose output, including FCP transcripts, to the stderr stream.
-H, --fcpHost=<hostname> Use FCP interface at host <hostname>, defaults to 127.0.0.1
-P, --fcpPort=<port> Use FCP interface at port <port>
-m, --mimetype=<mimetype> Specify the mimetype under which the key is to be inserted. If not specified, the program will try to guess a mimetype from the input file extension (if present), or from a 'file extension' suffix on the URI. If neither of these are present, default mimetype is text/plain.
-r, --priority=priority Sets priority to priority. Valid values are 0 (highest) to 6 (lowest). Default is 4.

 

FILES

none
 

ENVIRONMENT VARIABLES

FCP_HOST
Specifies the hostname of the FCP interface. This value will be used unless overridden with '-H'.
FCP_PORT
Specifies the port number of the FCP interface. This value will be used unless overridden with '-P'.

 

EXAMPLES

fcpput KSK@somekey filename.html
Insert file filename.html as URI KSK@somekey

someprog | fcpput KSK@somekey
Run a program, and insert its standard output as URI KSK@somekey

fcpput -H 192.168.10.4 CHK@fred.jpg somefile
Using the FCP service on host 192.168.10.4, insert somefile as a CHK, with the implied mimetype of image/jpeg. The final CHK URI under which this key can be later retrieved will be printed to stdout.
