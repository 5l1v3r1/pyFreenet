NAME

fcpget - Retrieve a single key from Freenet  

SYNTAX

fcpget [options] key_uri [filename]  

DESCRIPTION

fcpget is a simple command-line FCP client program for retrieving single keys from Freenet.  

OPTIONS

-h, --help
Print help information and exit
-V, --version
Print version number and exit
-v, --verbose
Enable very verbose output, including FCP transcripts, to the stderr stream.
-H, --fcpHost=hostname
Use FCP interface at host hostname, defaults to 127.0.0.1
-P, --fcpPort=port
Use FCP interface at port port
-r, --priority=priority
Sets priority to priority. Valid values are 0 (highest) to 6 (lowest). Default is 4.

 

FILES

none
 

ENVIRONMENT VARIABLES

FCP_HOST
Specifies the hostname of the FCP interface. This value will be used unless overridden with '-H'.
FCP_PORT
Specifies the port number of the FCP interface. This value will be used unless overridden with '-P'.

 

EXAMPLES

fcpget KSK@somekey
Retrieve key URI KSK@somekey from Freenet, and print its data to standard output.

fcpget KSK@somekey filename.html
Retrieve key URI KSK@somekey from Freenet, and save its data to filename.html
