NAME

fproxyproxy - a name-service-enabled http proxy service over fproxy

 

SYNTAX

fproxyproxy [options]  

DESCRIPTION

fproxyproxy is an http proxy server that sits on top of Freenet's FProxy server. It translates human-friendly freenet 'domain names' received from the client into proper freenet URIs, feeds these URIs into fproxy, and feeds FProxy's output back to the client.

Whereas FProxy is designed to be used as a web server, fproxyproxy is designed to be configured into the browser as an http proxy server. Fortunately, with excellent quick proxy-switching features of modern browsers (such as the SwitchProxy extension in Firefox), this becomes an easy and simple operation.

As an example of the use of fproxyproxy, if you had a 'freenet name services' record for the 'pyfcp' website, you would be able to type http://pyfcp into your browser, and be viewing the pyfcp freesite! (As long as you have set up fproxyproxy as an http proxy in your web browser).

fproxyproxy is a proof of concept for using secure, human-friendly freesite addresses within freenet, and is intended to be used alongside the fcpnames(1) utility.

If you want to access FProxy directly in a browser configured to use fproxyproxy as a proxy server, just enter the address http://fproxy.

 

OPTIONS

-h, --help
Print help information and exit
-V, --version
Print version number and exit
-v, --verbose
Enable very verbose output, including FCP transcripts, to the stderr stream.
-H, --fcpHost=hostname
Use FCP interface at host hostname, defaults to 127.0.0.1
-P, --fcpPort=port
Use FCP interface at port port
-p, --fproxyAddress=[hostname][:port]
Point fproxyproxy at the FProxy service at address hostname:port, default is 127.0.0.1:8888.
-L, --listenAddress=[hostname][:port]
Sets the address fproxyproxy listens on for client http connections, default is 127.0.0.1:8889.

 

FILES

none
 

ENVIRONMENT VARIABLES

FCP_HOST
Specifies the hostname of the FCP interface. This value will be used unless overridden with '-H'.
FCP_PORT
Specifies the port number of the FCP interface. This value will be used unless overridden with '-P'.

 

EXAMPLES

fproxyproxy -L localhost:2225 -p 192.168.1.5
Run fproxyproxy, listening for client connections on localhost:2225, and forwarding requests to an FProxy port at 192.168.1.5:8888 (note the default 8888 being applied to fproxy port).
