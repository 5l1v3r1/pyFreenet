NAME

freesitemgr - Insert or re-insert freesites into freenet

 

SYNTAX

freesitemgr [options] command [arguments]  

DESCRIPTION

freesitemgr is a simple command-line freenet client program for inserting freesites.

Use of this program is controlled by the command argument you give it.

run freesitemgr -h for a detailed help output.  

OPTIONS

-h, --help
Print help information and exit
-v, --verbose Enable very verbose output, including FCP transcripts, to the stderr stream.
-q, --quiet Run quietly
-l, --logfile location of logfile (default /home/david/updatesites.log)
-s, --single-files Insert each file of the freesite as single CHKs, and redirect to these from within the freesite manifest

 

COMMANDS

setup Create or modify the freesite config file (~/.freesites) interactively.

add Add a new freesite interactively

list [<name>] display a summary of all freesites, or a detailed report of one site if <name> given

remove <name> remove given freesite

update reinsert any freesites which have changed since they were last inserted

 

ENVIRONMENT VARIABLES

FCP_HOST
Specifies the hostname of the FCP interface. This value will be used unless overridden with '-H'.
FCP_PORT
Specifies the port number of the FCP interface. This value will be used unless overridden with '-P'.
