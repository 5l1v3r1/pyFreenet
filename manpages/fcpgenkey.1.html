NAME

fcpgenkey - generate a single Freenet SSK@/USK@ keypair

 

SYNTAX

fcpput [options]  

DESCRIPTION

fcpgenkey is a simple command-line FCP client program for generating new Freenet SSK@ or USK@ public/private keypairs.

The keypair generated is written to standard output stream, with the public key, then a newline, then the private key.

 

OPTIONS

-h, --help
Print help information and exit
-V, --version
Print version number and exit
-v, --verbose
Enable very verbose output, including FCP transcripts, to the stderr stream.
-H, --fcpHost=<hostname>
Use FCP interface at host <hostname>, defaults to 127.0.0.1
-P, --fcpPort=<port>
Use FCP interface at port <port>

 

FILES

none
 

ENVIRONMENT VARIABLES

FCP_HOST
Specifies the hostname of the FCP interface. This value will be used unless overridden with '-H'.
FCP_PORT
Specifies the port number of the FCP interface. This value will be used unless overridden with '-P'.

 

EXAMPLES

fcpgenkey > keys.txt
Generate a keypair and save it to keys.txt
